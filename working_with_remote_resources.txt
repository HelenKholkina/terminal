1) ping - чтобы узнать IP-адрес сайта. Синтаксис: ping адрес_сайта.ru 

2) wget - для скачивания файлов, веб-страниц и других ресурсов из интернета. 
Синтаксис: wget опции URL
Например, wget https://domain.ru/filename.tar.bz2 - скачать файл filename.tar.bz2
Опции: -O - новое_название_файла
       -i - название_текстового_файла_в_котором список URL, откуда скачивать

3) ssh - это протокол для установки связи между ПК и сервером, то есть для
удаленной работы с сервером. Как подключить:
       * запустить терминал
       * вводим: ssh имя_пользователя@адрес_сервера
       * вводим пароль
       * жмем Enter

Заменим стандартный 22-ой порт для безопасности:
ssh -p номер_порта имя_пользователя@_адрес_сервера

4) scp - копирование файлов по ssh между компами под управлением Linux.
Синтаксис: scp опции что_копируем куда_копируем
Примеры: а. scp myfile.dat pingvin@192.168.1.74:/home/pingvin - скопируем ssh 
локальный файл myfile.dat на комп с IP-адресом 192.168.1.74 в директорию
/home/pingvin - копирование файсла с локального на удаленный
         б. scp pingvin@192.168.1.74:/home/pingvin/myfile.dat . - копирование
с удаленного на локальный.

5) curl - это "утилита"(команда), которая имитирует действия пользователя
на страницах и других операций с URL-адресами

6) imap - протокол прикладного уровня для доступа к электронной почте

7) telnet - сетевая утилитаа, которая позволяет соединиться с удаленным портом любого
компа и установить канал связи. Используют для тестирования сети, проверки портов.
