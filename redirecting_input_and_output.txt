Данные, полученные в результате выполнения команды, можно перенаправить в файл. Например:

cat test1 test2 > test12  -  выведенные данные двух файлов отправились в файл test12.

* при дальнейшем перенапрвлении данных в test12 с помощью того же > приведет к перезаписи
поверх данных из test1 и test2. Для того, чтобы добавить данные к уже имеющимся, используют >>.

* можно записывать результаты команд, добавляя >> имя_файла_для_записи, после команды.
